# Player Is You
A 2D puzzle game inspired by Baba Is You. Created in Unity.

# Controls
|Key   |Description                        |
|------|-----------------------------------|
|Arrows|Menu navigation, character movement|
|Return|Select                             |
|Escape|Back, pause                        |
|D+L   |Reset progress (on start screen)   |
|R     |Restart level (on game screen)     |
